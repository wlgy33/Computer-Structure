# 00 자료구조 Data Structure

## 자료구조란?
자료(Data)의 집합으로 각 원소들이 정의된 규칙에 따라 정리되어 자료에 대한 처리를 효율적으로 수행할 수 있도록 표현한 것
여러 자료구조들이 존재하는데 어떤 경우에 무슨 자료구조를 사용해야 될지 기준이 있다.

### 자료구조 선택기준
- 처리시간
- 크기
- 활용 빈도
- 프로그램의 용이성
- 갱신의 정도

## 자료구조의 특징
1. 효율성
검색 알고리즘(Search Algorithm)을 수행할 때, 데이터의 양이 많다면 순차 검색(Linear Search)을 사용하는 것보다 이진 탐색(Binary Search)를 사용하는 것이 더 효율적이다. 이처럼 유형에 따라 사용할 자료구조의 유형이 달라진다.

2. 추상화
추상화란 대상에 대해 그것이 가져야 할 핵심적인 특징을 가지는 모델을 만드는 것이다.

3. 재사용성
자료구조를 설계할 때 특정 프로그램에서만 사용 가능하게 설계하지 않고 범용성 있게 설계한다.

## 자료구조의 분류
자료구조는 크게 선형 자료구조와 비선형 자료구조로 나뉜다. 선형 자료구조의 경우 데이터를 순차적으로 나열시킨 형태를 말하며, 비선형 자료구조는 하나의 자료 뒤에 여러 개의 자료가 존재할 수 있는 것을 말한다.
